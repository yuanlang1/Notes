## 1. 在创建表时添加约束

```sql
create table test(
                    id int not null,
                    name varchar(10) not null,
                    salary decimal(10,2)
                 );

例如:
insert into test
values (null,'TOM',1000); -> 错误,id不能为空

insert into test(id,salary)
values (1,1000); -> 错误,name为空
```

---

## 2. 在`alter table`时添加

```sql
alter table test
modify salary int not null; -> 如果表中已经存在null值的数据,无法添加not null
```
