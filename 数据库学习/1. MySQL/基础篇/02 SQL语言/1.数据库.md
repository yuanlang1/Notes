## 1.查看现有数据库  

```sql
show databases;
```

---

## 2.新建数据库  

```sql
create database <数据库名称>
```

---

## 3.选择数据库  

```sql
use <数据库名>
```  

---

## 4.从sql文件中引入数据库  

```sql
source <.sql文件路径>
```
